## Family Context

- Sydney, Australia (Woolworths for groceries)
- Two young kids — toddler-friendly is non-negotiable
- No pork, no alcohol in cooking, no chilli heat
- 30 min max hands-on time on weeknights
- Hidden vegetables are a core strategy

## Usage with Claude

These files are used as knowledge docs in a Claude Project for weekly meal planning, grocery list generation, and recipe development. See the project instructions within Claude for the full planning guidelines.
